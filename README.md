# C2C Online Shop
This application is an prototyping of SA teamwork. The topic is about virtual currency, c2c online shop, personal security and blockchain.

## Features
- Order Product
- Launch Product
- Manage Product
- Seacher Product
- Edit Profile
- Make Payment
- Withdraw Money
- Deposit Coins
